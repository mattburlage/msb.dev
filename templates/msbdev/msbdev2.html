My Development Principles

  • Security First
    In today's world, the value of data is ever increasing. With this in mind, any software system should consider security from the outset. From design through implementation and maintenance, security should be front and center. No system is worth a penny if it is insecure.
  • Ease of Use
    The number one goal of any system should, outside security, be to create a wonderful user experience for end users and admins alike. Users should be able to focus on their work, not on learning new systems. This can be accomplished by focusing the experience around the tasks that your users need to complete. All a user needs to know is what their end goal is, and the system should take care of putting the information they need in front of them or whomever needs it.
  • Embrace and Prepare for Change
    Business needs change regularly, and any successful enterprise must have software that can adjust with them. Software solutions, therefore, must be customizable and built to handle changing needs. This includes building them from day one in a way that makes later upgrades a breeze.
Projects
Agenda Management   React, Django, PostgreSQL, IIS
The Agenda Management software is designed to assist governments of all sizes in the preparation, approval, creation, publishing, recording, and archiving of official Board meetings. The system allows for an organization to allow different departments to enter their desired agenda items, which then flow through a completely customizable workflow for needed approvals. The system allows for the production of PDFs for each agenda item, the agenda list, the minutes, any supporting documents, and conversion of any attachments to PDF. Those PDFs and agendas are also uploaded to a public site to comply with any publication requirements that may exist.
Five Star Ordering   React, Django, PostgreSQL, PythonAnywhere
The Five Star ordering system is a fixed-rate restaurant online ordering platform that supports both pickup and delivery orders. It allows for address verification and integrates with Stripe payment processing. The platform allows new restaurants to be added with zero additional coding and can be completed as quickly as a few hours.
Position Control   Django, React, Microsoft SQL, IIS
The Position Control system is designed to handle the tracking and paperwork for all employee records. It handles all paperwork including new hires, job changes, termination, and more. It also handles the creation and budgeting of new positions. It allows approval on both the department and administration level. It also handles automatic creation of Active Directory user accounts and uses Active Directory for user authentication. The Finance department can use the system to calculate labor costs for individual departments or the whole county, and projects future budgets.
